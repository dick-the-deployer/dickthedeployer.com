Dick the Deployer

Dockerized Continuous Delivery

Built on Docker, Dick The Deployer gives you the ability to implement easy and scalable Continuous Delivery.

Dick the Deployer differs from other continuous integration applications in that it promotes scalability and is easy to adapt and maintain. Dick the Deployer performs pipelined builds using configuration stored inside project repository, presenting results near-real time on clean UI.

User Interface

The user interface is build around concepts familiar to every programmer, like groups and projects. You can manage those projects and groups, browse builds history and observe progress on each defined pipeline.

For details, see the Walkthrough.

Dick the Deployer ships with rich API enabling easy integration with existing tools.
